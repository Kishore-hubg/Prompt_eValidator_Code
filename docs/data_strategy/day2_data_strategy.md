# Day 2 Data Strategy

## Store
- users
- persona_assignments
- prompt_validations
- dimension_scores
- prompt_rewrites
- channel_usage
- feedback_events

## Required fields
- user_id / email
- persona_id
- channel (api, web, mcp, teams)
- original_prompt
- final_score
- issue_count
- improved_prompt
- created_at

## Why this matters
This supports analytics, persona benchmarking, and later human review.
