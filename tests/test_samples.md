# Sample prompts for quick testing

## Persona 0
Summarize the attached meeting transcript for an executive audience in 5 bullet points. Include top 3 decisions, owners, and next steps. Keep it under 150 words.

## Persona 1
Generate a Python 3.11 FastAPI endpoint for POST /users with SQLAlchemy, Pydantic validation, JWT auth, and pytest unit tests. Include edge cases for invalid payload, duplicate email, and empty request body. Return code in separate blocks.

## Persona 2
Create a sprint 15 leadership status report for fixed-bid engagement. Use a structured template with RAG status, top blockers, risk ranking, current velocity vs baseline, and recommended next actions.

## Persona 3
Based on the provided BRD Section 2 and stakeholder notes, extract only explicit functional requirements. Do not infer unstated requirements. Return a table with Requirement ID, Requirement, Source, and Business Rationale.

## Persona 4
Draft a short empathetic customer email for a delayed support ticket. Mention current status, next step, ETA if available, and a policy-compliant closing.
